Descrição:
	Gerencie temas de formas práticas, pré-Cadastre as configurações de seu tema e altere ele com apenas 1 clique.
	 - Pre-Cadastre seu temas e ative-os em apenas um clique
	 - Altere os arquivos controller diretamente no painel administrativo
	 - Altere os arquivos de linguagem diretamente no painel administrativo
	 - Altere os arquivos model diretamente no painel administrativo
	 - Altere o layout de seu tema diretamente no painel administrativo.

Instalação:
	1 - Extra o arquivo Manager_Layout.zip
	2 - Aplique o código do arquivo install.sql no seu banco de dados.
	3 - Acesse o painel administrativo de sua loja, vá em Sistema > Usuários > Grupos de Usuários > Clique em editar.
	4 - Dê permissões de acesso e  modificação para:
		4.1 -  appaearance/controllereditor
		4.2 - appaearance/languageeditor
		4.3 - appaearance/modeleditor
		4.4 - appaearance/themeditor
		4.5 -  layout/layout
